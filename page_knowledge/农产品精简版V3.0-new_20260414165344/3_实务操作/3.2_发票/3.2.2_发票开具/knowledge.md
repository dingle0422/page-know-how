- **3.2.2.2_批量开票说明**: 3.2.2.2.  批量开票说明
（1）支持“农产品收购发票”的批量开具。需要提醒的是，开具农产品收购发票会影响数电票的可用额度。发票开具后，“可用发票额度”会根据开具的收购金额自动减少。
（2）暂不支持“自产农产品销售发票”的批量开具。
- **3.2.2.3_开票规范**: 3.2.2.3.  开票规范
（1）免税发票：税率栏显示“免税”，不得显示“0%”。开具免税发票意味着放弃开具专用发票的权利。
（2）应税发票：根据纳税人类别和货物类型，选择1%、3%、9%、13% 等适用税率。
（3）农产品销售发票（数电票）：必须通过特定业务模块开具，并选择“自产农产品销售”标签，否则购买方无法抵扣。
