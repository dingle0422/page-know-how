3.2.1. 发票勾选
在数电票试点后，农产品发票的进项抵扣必须先通过电子税务局进行“抵扣勾选”（税控票时期企业通过增值税发票综合服务平台勾选发票，目前仍在使用税控盘开具发票的企业极少，基本上都是通过电子税务局开具、勾选发票），不能直接填表。具体操作路径：

|  |  |  |  |
| --- | --- | --- | --- |
| **发票类型** | | **操作路径** | **说明** |
| 农产品销售发票 | 左上角带“自产农产品销售”标签的免税数电票 | 【抵扣勾选】→ 直接勾选 | 与常规专票操作相同。 |
| 税控系统开具的自产农产品免税发票 | 【待处理农产品发票】→【自产农产品销售发票】→ 处理 → 再回到【抵扣勾选】 | 需先预处理，确认是否为自产。 |
| 小规模纳税人开具的3%农产品专票 | | 【待处理农产品发票】→【从小规模处购进的3%农产品专票】→ 处理（选择“按票面金额和基础扣除率”）→ 再回到【抵扣勾选】 | 预处理时需选择抵扣方式：  ①按票面金额×9%计算抵扣（推荐）  ②按票面税额抵扣（不推荐） |
| 税务机关代开的农产品免税发票（个人自产） | | 【抵扣勾选】→【代开农产品发票录入】→ 补录信息 → 再回到【抵扣勾选】 | 补录销售方信息、发票代码、号码、金额等。 |
| 农产品收购发票 | | 【抵扣勾选】→ 直接勾选 | 由购买方自行开具，系统可识别。 |
| 1%农产品专票 | | 【抵扣勾选】→ 直接勾选 | 按票面税额抵扣，无需特殊处理。 |
