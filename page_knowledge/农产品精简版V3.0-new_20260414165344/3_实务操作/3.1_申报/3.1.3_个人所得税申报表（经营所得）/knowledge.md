3.1.3.  个人所得税申报表（经营所得）
个体工商户、个人独资企业、合伙企业等申报经营所得的主体，从事农林牧渔业（四业）取得的所得，免征个人所得税（非个人所得税核定征收的情形），需要在申报经营所得时，按照以下流程申报：
进入“预缴纳税申报”（年报点击“年度汇缴申报”），填写公共信息后，双击投资者信息进入报表，点击“减免税额”右侧“小红笔”图标，附表中选择“减免事项”和“减免性质”，填写减免税额后保存就可以了。
