## 本章节内容

2.1.2.1. 免税场景

|  |  |  |  |
| --- | --- | --- | --- |
| **业务场景** | **免税条件** | **开票规定** | **风险提示** |
| （1）农业生产者销售自产农产品 | 直接从事植物种植、动物饲养、捕捞的单位和个人销售其初级农产品。 | 开具免税普通发票，非自然人可以放弃优惠开具专票，如个体工商户、企业、农民专业合作社等，小规模纳税人适用1%/3%，一般纳税人适用9%。自然人不得开具增值税专用发票。 | 享受免税优惠必须是“自产”，外购再销售不享受；  “公司+农户”模式，2026年起政策未明确，需与主管税局确认。 |
| （2）流通环节销售蔬菜（批发零售环节） | 从事蔬菜批发、零售的纳税人。 | 开具免税普通发票，非自然人可以放弃优惠开具专票（小规模纳税人1%/3%，一般纳税人9%），自然人不得开具增值税专用发票。 | 蔬菜的范围参照《财政部 国家税务总局关于免征蔬菜流通环节增值税有关问题的通知》（财税[2011]137号）的附件《蔬菜主要品种目录》，各种蔬菜罐头不享受；  若放弃免税，需书面声明，36个月内不得再申请（小规模纳税人除外）。 |
| （3）流通环节销售部分鲜活肉蛋产品（批发零售环节） | 从事批发、零售的纳税人销售猪、牛、羊、鸡、鸭、鹅的鲜、冷、冻肉及组织，以及鸡蛋、鸭蛋、鹅蛋。 | 开具免税普通发票，非自然人可以放弃优惠开具专票（小规模纳税人1%/3%，一般纳税人9%），自然人不得开具增值税专用发票。 | 免征增值税的鲜活肉产品，是指猪、牛、羊、鸡、鸭、鹅及其整块或者分割的鲜肉、冷藏或者冷冻肉，内脏、头、尾、骨、蹄、翅、爪等组织。  免征增值税的鲜活蛋产品，是指鸡蛋、鸭蛋、鹅蛋，包括鲜蛋、冷藏蛋以及对其进行破壳分离的蛋液、蛋黄和蛋壳。  仅限列举的几种家畜家禽和蛋类，鹌鹑蛋、鸵鸟蛋等不享受；不包括国家珍贵、濒危野生动物。具体参考《国家税务总局关于免征部分鲜活肉蛋产品流通环节增值税政策的通知》财税[2012]75号。 |
| （4）批发和零售种子、种苗 | 税收优惠有效期至2027年12月31日。 | 开具免税普通发票，非自然人可以放弃优惠开具专票（小规模纳税人1%/3%，一般纳税人9%），自然人不得开具增值税专用发票。 | 具体参考《财政部 国家税务总局关于若干农业生产资料征免增值税政策的通知》（财税[2001]113号）。 |
| （5）生产销售和批发、零售有机肥 | 税收优惠有效期至2027年12月31日，产品需符合有机肥料、生物有机肥等国家标准。 | 开具免税普通发票，非自然人可以放弃优惠开具专票（小规模纳税人1%/3%，一般纳税人9%），自然人不得开具增值税专用发票。 | 有机肥不属于农产品，但是比较容易混淆。享受有机肥税收优惠需单独核算有机肥销售额，并向税务机关备案。 |
| （6）承担粮食收储任务的国有粮食购销企业销售粮食 | —— | 按不免税方式开具增值税专用发票，购买方凭票抵扣。 | 属于特殊规定，免税业务但可开专票。 |
| （7）纳税人将国有农用地出租给农业生产者用于农业生产 | —— | 需开具免税普通发票。 | —— |
| （8）“公司+农户”模式销售畜禽（2026年1月1日前） | 属于农业生产者销售自产农产品。 | 需开具免税普通发票。 | 2026年1月1日起政策衔接不明，各地口径可能不同，需与主管税务机关确认。 |
| （9）对农民专业合作社销售本社成员生产的农业产品 | 视同农业生产者销售自产农业产品免征增值税。 | 需开具免税普通发票。 | 2026年增值税法执行后，有了优惠期间，自2026年1月1日至2027年12月31日，免征增值税。 |

注1：进口农产品在批发、零售环节销售，如果属于批发零售环节免征增值税的农产品类别，则可以享受免税政策。
注2：销售从非社员的农户收购的农产品（不含鲜活肉蛋产品、蔬菜、种子、种苗），再销售时，不能享受免征增值税优惠：若为一般纳税人，应按9%税率缴纳增值税；若为小规模纳税人，按照3%征收率缴纳，自2023年1月1日至2027年12月31日，减按1%征收率。
