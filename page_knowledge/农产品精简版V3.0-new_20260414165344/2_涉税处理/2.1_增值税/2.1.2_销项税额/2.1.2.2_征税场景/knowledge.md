## 本章节内容

2.1.2.2. 征税场景

|  |  |  |
| --- | --- | --- |
| **征税场景** | **一般纳税人** | **小规模纳税人** |
| （1）销售初级农产品（非免税） | 9% | 3% (2027.12.31前可减按1%) |
| （2）销售深加工农产品（如罐头、精制茶、中成药） | 13% | 3% (2027.12.31前可减按1%) |
| （3）销售非农产品（如种植土等容易与农产品混淆的产品） | 13% | 3% (2027.12.31前可减按1%) |
| （4）销售初加工大米连同包装物 | 9% | 3% (2027.12.31前可减按1%) |

注1：区分初级农产品与深加工产品是关键。例如：毛茶（9%/免税），精制茶（13%）；鲜奶（9%/免税），酸奶、调制乳（13%）；中药材（9%/免税），中成药（13%）。初级农产品严格在《财政部 税务总局关于增值税征税具体范围有关事项的公告》（财政部 税务总局公告2026年第9号）附件1《适用9%增值税税率货物范围注释》一、农产品的正列举范围内。深加工则不在列举内，或明确排除。
注2：外购生猪直接转售（鲜活）：一般纳税人可选 9% 计税（开专票）或选流通环节鲜活肉蛋免税（开普票）；小规模纳税人通常免税或选征收率3%（2027年底之前征收率为1%）；外购后屠宰分割再销售：仍属鲜活肉蛋，可享受流通环节免税；外购后深加工（如熟制、腌制），按13%计税。
注3：个体工商户既可以是小规模纳税人，也可以登记为一般纳税人。
