## 本章节内容

2.3. 其他税种

|  |  |  |
| --- | --- | --- |
| **税种** | **核心规则** | **风险提示** |
| （1）印花税 | 免征：农民、家庭农场、农民专业合作社、农村集体经济组织、村民委员会销售农产品书立的买卖合同。  不征：个人（含个体户）书立的动产买卖合同。  征税：除上述主体外，企业之间签订的农产品买卖合同，按价款的万分之三缴纳。 | 计税依据为不含增值税的合同金额。若合同未单独列明增值税，则需按全额计税。 |
| （2）个人所得税 | 暂不征收：个人、个体户、个人独资企业、合伙企业从事种植业、养殖业、饲养业、捕捞业（“四业”）取得的所得。  核定征收除外：实行核定征收的个人独资企业和合伙企业，不能享受该优惠。 | 支付给农户的农产品收购款或委托饲养费，**无需**代扣代缴个人所得税。 |
| （3）城镇土地使用税 | 免征：直接用于农、林、牧、渔业的生产用地（包括经营采摘、观光农业的种植、养殖、饲养土地）。 | 办公用地、加工厂用地等不属于生产用地，需缴纳。 |
| （4）环保税 | 不缴纳：对畜禽养殖废弃物进行综合利用和无害化处理，不属于直接排放。  需缴纳：达到省级政府确定的规模标准且有污染物排放口。 | 禽畜养殖业水污染物应纳税额 = 月均存栏量 ÷ 污染当量值 × 单位税额；  月均存栏量 = (月初存栏量 + 月末存栏量) / 2。 |
| （5）烟叶税（烟叶特指烤烟叶和晾晒烟叶） | 纳税人：在我国境内收购烟叶的单位。  计税依据：收购价款 × (1+10%)。价外补贴统一按10%计算。  税率：20%。  纳税地点：烟叶收购地。 | 烟叶税计入农产品买价，作为计算增值税进项抵扣的基础。另外，烟叶不属于消费税范围，不需要缴纳消费税。 |
| （6）耕地占用税 | 耕地占用税以纳税人实际占用的属于耕地占用税征税范围的土地面积为计税依据，按应税土地当地适用税额计税，实行一次性征收。  耕地占用税计算公式为：应纳税额=应税土地面积×适用税额。 | 占用园地、林地、草地、农田水利用地等农用地建设建筑物、构筑物或者从事非农业建设的，需要依法缴纳耕地占用税；如果是建设直接为农业生产服务的生产设施，则不需要缴纳耕地占用税。 |
