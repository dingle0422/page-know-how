## 本章节内容

2.1.2.2. 发票类型与抵扣规则汇总表

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| **发票类型** | | **销售方身份/业务性质** | **购买方抵扣规则** | **关键条件与风险提示** |
| 农产品销售发票 | （1）数电免税普票 | 自产自销 | 按票面金额×9%计算抵扣【票面金额通常是指不含税金额，但是免税情形下，票面金额和含税金额也是相等的，不需要做价税分离】 | 发票左上角必须有 “自产农产品销售” 标签，否则无法勾选抵扣。这是最大风险点！ |
| （2）税控系统开具的免税普票 | 自产自销 | 按票面金额×9%计算抵扣 | 无标签要求，但需通过“待处理农产品发票”模块预处理后勾选。需确保业务实质为自产。 |
| （3）税务机关代开的免税普票 | | 自然人自产自销 | 按票面金额×9%计算抵扣 | 若为数电票，同样需要“自产农产品销售”标签；非数电票则需在“代开农产品发票录入”模块补录信息。 |
| （4）农产品收购发票 | | 购买方向农业生产者个人收购自产农产品时自行开具 | 按票面金额×9%计算抵扣 | 发票需注明“收购”或“农产品收购”字样。一般只限本县（市、区）范围内开具。 |
| （5）3%增值税专用发票 | | 小规模纳税人（不区分是否自产） | 按票面金额×9%计算抵扣 | 需先在“待处理农产品发票”模块处理。这是最有利的专票类型，可实现“低征高扣”。 |
| （6）1% 增值税专用发票 | | 小规模纳税人（不区分是否自产） | 只能按票面注明的1%税额抵扣，不能按9%计算抵扣 | 常见误区：很多企业误以为1%专票也能按9%抵扣，这是严重错误！ |
| （7）9%增值税专用发票 | | 一般纳税人（不区分是否自产） | 按票面注明的9%税额进行抵扣 | 常规凭票抵扣，无特殊计算。 |
| （8）海关进口增值税专用缴款书 | | 进口农产品 | 按缴款书注明的增值税额抵扣 | —— |
| （9）流通环节免税普票 | | 批发、零售免税蔬菜/鲜活肉蛋/种子种苗的纳税人 | 不得抵扣 | 这是最容易被误抵扣的发票！此类发票用于已享受免税的流通环节，进项税不得抵扣。 |
| （10）带税率的普通发票 | | 任何纳税人（如小规模纳税人开具1%/3%普票） | 不得抵扣 | 普通发票无论税率多少，均不能作为进项抵扣凭证。必须取得专用发票、农产品销售发票（自产自销）、农产品收购发票、海关进口增值税专用缴款书才可以抵扣进项税额。 |
