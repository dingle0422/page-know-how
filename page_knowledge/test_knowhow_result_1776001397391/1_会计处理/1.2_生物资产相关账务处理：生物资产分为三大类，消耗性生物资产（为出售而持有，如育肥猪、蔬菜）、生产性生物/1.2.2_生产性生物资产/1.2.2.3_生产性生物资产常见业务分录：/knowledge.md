## 本章节内容

1.2.2.3. 生产性生物资产常见业务分录：

|  |  |  |
| --- | --- | --- |
| **业务场景** | **会计分录 (执行企业会计准则)** | **说明** |
| （1）外购产畜/果树（未成熟） | 借：生产性生物资产—未成熟生产性生物资产   贷：银行存款 | 达到预定生产经营目的前 |
| （2）发生饲养、管护等成本 | 借：生产性生物资产—未成熟生产性生物资产   贷：原材料/应付职工薪酬/银行存款 | 资本化 |
| （3）达到预定生产经营目的（如开始产奶） | 借：生产性生物资产—成熟生产性生物资产  贷：生产性生物资产—未成熟生产性生物资产 | 从“未成熟”转入“成熟” |
| （4）成熟后发生饲料、防疫、折旧等支出 | 借：农业生产成本/生产成本  贷：原材料/应付职工薪酬/生产性生物资产累计折旧/银行存款等 | 支出费用化，归集为产品成本 |
| （5）收获农产品验收入库（如牛奶、鸡蛋） | 借：农产品/库存商品   贷：农业生产成本/生产成本 | 将归集的成本转入“农产品”或“库存商品” |
| （6）计提折旧 | 借：农业生产成本/生产成本/管理费用   贷：生产性生物资产累计折旧 | 根据资产用途确认借方科目 |
| （7）出售生产性生物资产（如淘汰奶牛） | 借：银行存款/应收账款等         生产性生物资产累计折旧         生产性生物资产减值准备（若涉及）  贷：生产性生物资产         资产处置损益（或借方） | 差额计入“资产处置损益”（小企业准则计入“营业外收入”或“营业外支出”） |
| 自行繁殖小奶牛 | 归集母牛孕期及接产成本：     借：农业生产成本—繁育成本     贷：原材料/应付职工薪酬等  小牛出生时：     - 作为消耗性（育肥出售）：       借：消耗性生物资产—犊牛       贷：农业生产成本—繁育成本     - 作为生产性（留种产奶）：       借：生产性生物资产—未成熟(犊牛)       贷：农业生产成本—繁育成本  后续饲养小牛时：   - 作为消耗性（育肥出售）：       借：消耗性生物资产—犊牛       贷：原材料—饲料              应付职工薪酬/银行存款等   - 作为生产性（留种产奶）：       借：生产性生物资产—未成熟(犊牛)       贷：原材料—饲料              应付职工薪酬              银行存款 | 小牛成本为分娩过程中发生的必要支出 |
| 生产性生物资产死亡 | 发现时：     借：待处理财产损溢             生产性生物资产累计折旧     贷：生产性生物资产            应交税费—应交增值税（进项税额转出）（仅当管理不善造成时需做，否则可省略）  批准后处理：     借：其他应收款（如有保险赔款或责任人赔款）         营业外支出（用于非正常损失，如因病死亡）         管理费用（如因管理不善造成）     贷：待处理财产损溢 | 损失原因需清晰，以区分责任 |
