## 本章节内容

1.2.4. 特殊业务场景账务处理及风险

|  |  |  |  |
| --- | --- | --- | --- |
| **业务场景** | **会计分录** | **说明** | **风险提示与应对策略** |
| （1）“买一赠一”方式购进货物（如购油送挂面） | 借：库存商品—食用油         库存商品—挂面        应交税费—应交增值税（进项税额）  贷：应付账款/银行存款等 | 将总价按公允价值在正品与赠品间分摊 | 风险：分摊比例不合理，导致成本不实。  策略：以供应商发票或市场价为基准，采用公允价值比例法分摊。 |
| （2）“买一赠一”方式进行销售（如送大米促销） | 借：银行存款   贷：主营业务收入—商品A          主营业务收入—商品B | 将总销售额按商品公允价值比例分摊确认收入，不属于无偿捐赠，不视同销售。 | 风险：错误地将赠品做视同销售，多交税。  策略：确保在合同或发票中注明“买一赠一”的组合销售模式。 |
| （3）采购农产品收到相同货物的赠品（如采购76000个苹果，供应商又赠送了32个） | 借：库存商品 (76000个的总价，入库76032个)      应交税费—应交增值税（进项税额）  贷：应付账款/银行存款等 | 总成本不变，重新计算入库单价 | —— |
| （4）销售生产过程中的边角料/下脚料 | 借：银行存款   贷：其他业务收入       应交税费—应交增值税(销项税额) | 相关的成本已经隐含在正品的成本中，由于下脚料相对于正品来说金额较低，按照会计核算的重要性原则，一般不需要追溯调减正品的生产成本。实务中，若企业担心无匹配的成本，计入营业外收入也是可以的。 | 风险：漏记收入或适用错误税率（如农产品下脚料可能适用9%而非13%）。  策略：确认下脚料是否属于农产品范围，准确选择税率。 |
| （5）原材料入库后挑选，分不同等级入库 | 领料时：  借：生产成本  贷：原材料  发生人工成本时：  借：生产成本  贷：应付职工薪酬等   完工入库时：  借：原材料-A1等级         原材料-A2等级  贷：生产成本 | 将此过程视为加工过程，结转相关费用到原材料成本中，成本按合理方法（如售价法）分摊。 | 策略：建立合理的成本分摊标准（如售价、重量、数量），并保持一贯性。 |
| （6）以前年度进项税未转出被税局查补（追溯调整） | ①进项税额转出：  借：以前年度损益调整  贷：应交税费-增值税检查调整  ②进项税额转出补缴增值税：  借：应交税费-增值税检查调整  贷：应交税费-未交增值税  ③按照实际补缴的增值税计算需要补缴的附加税费：  借：以前年度损益调整  贷：应交税费-应交城建税         应交税费-应交教育费附加         应交税费-应交地方教育费附加等  ④进项税额转出部分依法可追补扣除：  借：应交税费-应交企业所得税  贷：以前年度损益调整  ⑤调整留存收益：  借：利润分配-未分配利润         盈余公积-法定公积金  贷：以前年度损益调整  ⑥实际补缴增值税：  借：应交税费-未交增值税         应交税费-应交城建税等         营业外支出（滞纳金+罚款）  贷：银行存款 | 企业会计准则需要追溯调整，小企业会计准则采用未来适用法，计入当期损益。 | 风险：税务稽查发现后，不仅补税，还有滞纳金和罚款。  策略：定期自查，确保用于免税/简易计税项目的进项税及时转出。 |
| （7）“公司+农户”模式养殖蛋鸡 | ①移交蛋鸡给农户：     借：生产性生物资产—蛋鸡(农户)     贷：生产性生物资产—蛋鸡(场部)  ②发放饲料/兽药：     借：消耗性生物资产—养殖成本     贷：原材料  ③支付农户服务费：     借：消耗性生物资产—养殖成本     贷：银行存款  ④鸡蛋入库：     借：库存商品—鸡蛋     贷：消耗性生物资产—养殖成本  ⑤销售鸡蛋/淘汰鸡：     确认收入并结转成本。 | 农户提供的仅是劳务，所有权始终归公司，会计上按自有资产核算。 | —— |
| 集团内销售免税农产品，合并报表抵销（乙方将农产品销售给集团内甲方） | ①甲方采购的农产品未对外销售，仍为集团内存货：  借：主营业务收入 (乙方确认的内部收入)  贷：主营业务成本 (乙方内部成本)        存货 (内部未实现利润)  借：存货 (甲方已抵扣的进项税)  贷：递延收益 (未来实现销售时转入损益)  ②甲方采购的农产品已对外实现销售（集团外）：  借：主营业务收入（乙方确认的内部销售收入）  贷：主营业务成本（乙方结转的内部销售成本）  主营业务成本（补回甲方抵扣的进项税对应的成本差额） | 在编制合并财务报表并抵销出售方对有关产品的未实现内部销售损益与购入方相应的存货账面价值时，该部分因增值税进项税额产生的差额在合并财务报表中可以确认为一项递延收益，并随着后续产品实现向第三方销售时再转入当期损益。 | 此处理应遵循《监管规则适用指引——会计类第1号》，确保合并报表层面正确反映增值税影响。 |
