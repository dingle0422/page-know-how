## 本章节内容

3.2.1.1. 常见申报异常及处理

|  |  |  |
| --- | --- | --- |
| **异常提示** | **原因分析** | **处理建议** |
| “行业为房屋建筑业、房地产业，《附列资料二》第6行不应填报” | 部分地区税务局设置了内部校验，认为该行业不应抵扣农产品。 | ①确认业务真实合规后，可忽略提示继续申报（提示类校验）。  ②若为强制校验，可强制申报，但可能导致开票系统被锁，需联系税务局解锁。  ③建议与主管局沟通，将企业纳入可抵扣行业清单。 |
| “附表二第6栏税额XX应等于农产品核定扣除增值税进项税额计算表（汇总表）中当期允许抵扣税额XX” | 企业被认定为核定扣除企业，但附表二第6栏填写了其他金额。 | ①检查是否属于核定扣除范围。  ②若属于，第6栏必须等于核定计算表合计数，且不得包含发票税额。  ③若部分产品核定扣除、部分非核定，需到税务局大厅申报。 |
| “农产品进项税额填报异常（第6栏、第8a栏）” | 税局票表比对校验不通过，可能数据填报有误。 | ①核对发票勾选数据与申报表是否一致。  ②确认2026年后是否错误填写了第8a栏。  ③数据正确可继续申报；错误则修改后重新申报。 |
| 农产品收购发票或者销售发票税额比对不通过 | 《增值税纳税申报表附列资料二》第6栏“农产品收购发票或者销售发票”的税额，以及《农产品核定扣除进项税额计算表（汇总表）》中的“本期允许抵扣进项税额”合计数不一致。 | 如果数据不相等，建议核对并修正申报表中的相关数据，确保准确无误后重新提交。 |
