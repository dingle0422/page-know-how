## 本章节内容

3.3.1. 数电票特定业务标签：在数电票系统中，农产品业务需使用特定业务模块开具。

|  |  |  |  |
| --- | --- | --- | --- |
| **特定业务标签** | **适用场景** | **操作要点** | **风险提示** |
| 自产农产品销售 | 农业生产者（企业、合作社、农户）销售自产的初级农产品。 | 1. 必须经税务机关授权，若选项缺失，联系主管局完成“业务归类”。  2. 开具时选择此标签，税率栏选“免税”。  3. 发票左上角自动生成“自产农产品销售”标识。 | 未使用此标签开具的免税数电票，购买方无法抵扣进项税。 |
| 农产品收购 | 企业向农业生产者个人收购自产免税农产品时，由购买方自行开具；收购非自产则不能开具。 | 1. 需经税务机关授权。  2. 开具时勾选“自然人”，并填写其证件类型和号码（否则会报错“证件类型和证件号码不能为空”）。  3. 税率栏选“免税”。 | 1. 仅限本县（市、区）范围内使用，跨区域收购请让农户代开普票。  2. 不得对非自产农产品（如从商贩处收购）开具收购发票。 |
