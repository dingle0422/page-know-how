【表格内容】
第1项：征税场景：（1）销售初级农产品（非免税）；一般纳税人：9%；小规模纳税人：3% (2027.12.31前可减按1%)。
第2项：征税场景：（2）销售深加工农产品（如罐头、精制茶、中成药）；一般纳税人：13%；小规模纳税人：3% (2027.12.31前可减按1%)。
第3项：征税场景：（3）销售非农产品（如种植土等容易与农产品混淆的产品）；一般纳税人：13%；小规模纳税人：3% (2027.12.31前可减按1%)。
第4项：征税场景：（4）销售初加工大米连同包装物；一般纳税人：9%；小规模纳税人：3% (2027.12.31前可减按1%)。
注：区分初级农产品与深加工产品是关键。例如：毛茶（9%/免税），精制茶（13%）；鲜奶（9%/免税），酸奶、调制乳（13%）；中药材（9%/免税），中成药（13%）。
