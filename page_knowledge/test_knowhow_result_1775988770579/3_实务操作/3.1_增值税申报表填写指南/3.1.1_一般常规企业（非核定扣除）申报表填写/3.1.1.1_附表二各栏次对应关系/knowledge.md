3.1.1.1. 附表二各栏次对应关系

|  |  |  |  |
| --- | --- | --- | --- |
| **栏次** | **栏次名称** | **适用发票类型** | **填写说明与风险提示** |
| 第2栏 | 本期认证相符且本期申报抵扣 | 1. 增值税专用发票（9%、1%税率）  2. 海关进口增值税专用缴款书 | 按票面注明的税额直接填写。**注意**：1%专票只能按票面税额填此栏，不得按9%计算后填第6栏。 |
| 第6栏 | 农产品收购发票或者销售发票 | 1. 农产品收购发票  2. 自产自销免税农产品普通发票（数电票需有标签）  3. 小规模纳税人开具的3%专票（按计算抵扣）  4. 税务机关代开的自产农产品免税发票 | **核心栏次**：填写按买价×9%计算出的进项税额。  **风险**：流通环节免税普票、1%专票、带税率普票均**不得**填入此栏。 |
| 第8a栏 | 加计扣除农产品进项税额 | 仅限2026年1月1日之前的业务 | 用于生产13%税率货物的农产品，在生产领用当期加计扣除1%。**2026年起此栏不再使用**。 |
| 第14栏 | 进项税额转出额 | 已抵扣但用于免税项目 | 填写转出的税额。 |
| 第15栏 | 进项税额转出额 | 已抵扣但用于不得抵扣项目（集体福利、个人消费） | 填写转出的税额。 |
| 第16栏 | 进项税额转出额 | 已抵扣但出现非正常损失 | 填写转出的税额。 |
| 第17栏 | 进项税额转出额 | 已抵扣但用于简易计税方法征税项目 | 填写转出的税额。 |
| 第23b栏 | 其他应作进项税额转出的情形 | 取得红字发票、税务稽查调增转出、不得抵扣非应税交易项目、年度清算调整、长期资产不得抵扣的进项税额 | 填写红字发票对应的税额或其他应转出额。  从2026年1月所属期开始，附表二23b“其他应作进项税额转出的情形”栏次不能直接填写了，点击该栏次后会有弹框，在弹框中填写数据后，小计栏次的数据将自动提取到附表二23b栏了。 |
| 第31栏 | 农产品收购发票或者销售发票 | 未申报抵扣的农产品发票 | 当企业已取得发票但暂不抵扣时，填入此栏；待实际抵扣时，再从第6栏抵扣（以单张发票为单位，不可拆分）。 |
