## 本章节内容

3.3.3. 发票开具常见问题与处理

|  |  |  |
| --- | --- | --- |
| **问题场景** | **正确处理方式** | **错误操作及风险** |
| （1）自产农产品销售，开票时税率栏选了“0%” | 应作废或红冲，重新开具税率栏为“免税”的发票。 | “0%”适用于出口业务，用于内销免税业务属于不合规发票，购买方无法抵扣。 |
| （2）收到对方开具的“0%”税率农产品发票 | 退回对方，要求重开“免税”发票。 | 不可作为进项抵扣凭证。 |
| （3）农产品收购发票信息错误（如姓名、身份证号）且已跨月 | 1. 由开票方（购买方）发起红冲，开具红字数电发票。  2. 在红冲当期做进项税额转出（填写附表二第23b栏）。  3. 重新开具正确的收购发票，重新勾选抵扣。 | 若未转出，会导致少缴税款；若金额错误未调整，成本核算不准。 |
| （4）已抵扣的农产品发票需要部分红冲 | 1. 购买方开具红字信息表/确认单。  2. 对方确认后开具红字发票。  3. 购买方在红字信息表通过的当期做进项税额转出（附表二第23b栏）。 | 转出时间错误（如在抵扣当期转出）会导致申报不符。 |
| （5）农产品收购发票错误开具为“0%”或“9%” | 红冲后重开，税率栏必须为“免税”。 | 税率错误属于不合规发票，不得作为税前扣除凭证。 |
| （6）跨区域收购农产品，无法开具收购发票 | 要求农户到当地税务机关代开免税普通发票，企业凭票入账并计算抵扣（非数电票需补录信息）。 | 不得强行开具本县（市、区）的收购发票。 |
| （7）销售方有自产能力，但未使用“自产农产品销售”标签开具数电票 | 联系销售方红冲重开，必须使用特定业务标签。 | 购买方无法勾选抵扣进项税。 |
| （8）流通环节免税蔬菜/鲜活肉蛋，客户要求开专票 | 告知客户此业务免税，无法开具专票； 若客户坚持要专票，企业可书面声明放弃免税，按9%或3%缴税后开具专票（36个月内不得再免税）。 | 擅自开具“免税”专票属于违规开票。 |
| （9）开具农产品发票失败，提示：“农产品收购发票，证件类型和证件号码不能为空” | 在开票界面（例如【蓝票开具】）中，找到并勾选“自然人”选项；然后，选择“农产品收购”作为特定业务；填写自然人的详细信息，包括证件类型和证件号码；完成这些步骤后，即可正常开具发票。 | —— |
