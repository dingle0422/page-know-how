## 本章节内容

2.1.3.1. 免征增值税的核心场景

|  |  |  |  |
| --- | --- | --- | --- |
| **业务场景** | **免税条件** | **开票规定** | **风险提示** |
| （1）农业生产者销售自产农产品 | 直接从事植物种植、动物饲养、捕捞的单位和个人销售其初级农产品。 | 开具免税普通发票，非自然人可以放弃优惠开具专票，自然人不得开具增值税专用发票。 | 享受免税优惠必须是“自产”，外购再销售不享受；  “公司+农户”模式，2026年起政策未明确，需与主管税局确认。 |
| （2）流通环节销售蔬菜 | 从事蔬菜批发、零售的纳税人。 | 开具免税普通发票，非自然人可以放弃优惠开具专票，自然人不得开具增值税专用发票。 | 1范围参照《财政部 国家税务总局关于免征蔬菜流通环节增值税有关问题的通知》（财税[2011]137号）的附件《蔬菜主要品种目录》，各种蔬菜罐头不享受；  若放弃免税，需书面声明，36个月内不得再申请。 |
| （3）流通环节销售部分鲜活肉蛋产品 | 从事批发、零售的纳税人销售猪、牛、羊、鸡、鸭、鹅的鲜、冷、冻肉及组织，以及鸡蛋、鸭蛋、鹅蛋。 | 开具免税普通发票，非自然人可以放弃优惠开具专票，自然人不得开具增值税专用发票。 | 仅限列举的几种家畜家禽和蛋类，鹌鹑蛋、鸵鸟蛋等不享受；  不包括国家珍贵、濒危野生动物。 |
| （4）批发和零售种子、种苗 | 税收优惠有效期至2027年12月31日。 | 开具免税普通发票，非自然人可以放弃优惠开具专票，自然人不得开具增值税专用发票。 | —— |
| （5）生产销售和批发、零售有机肥 | 税收优惠有效期至2027年12月31日，产品需符合有机肥料、生物有机肥等国家标准。 | 开具免税普通发票，非自然人可以放弃优惠开具专票，自然人不得开具增值税专用发票。 | 有机肥不属于农产品，但是比较容易混淆。享受有机肥税收优惠需单独核算有机肥销售额，并向税务机关备案。 |
| （6）承担粮食收储任务的国有粮食购销企业销售粮食 | —— | 按不免税方式开具增值税专用发票，购买方凭票抵扣。 | 属于特殊规定，免税业务但可开专票。 |
| （7）纳税人将国有农用地出租给农业生产者用于农业生产 | —— | 需开具免税普通发票。 | —— |
| （8）“公司+农户”模式销售畜禽（2026年1月1日前） | 属于农业生产者销售自产农产品。 | 需开具免税普通发票。 | 2026年1月1日起政策衔接不明，各地口径可能不同，需与主管税务机关确认。 |

注：进口农产品在批发、零售环节销售，如果属于批发零售环节免征增值税的农产品类别，则可以享受免税政策。
