## 本章节内容

2.1.3.2. 征税场景下的适用税率

|  |  |  |
| --- | --- | --- |
| **征税场景** | **一般纳税人** | **小规模纳税人** |
| （1）销售初级农产品（非免税） | 9% | 3% (2027.12.31前可减按1%) |
| （2）销售深加工农产品（如罐头、精制茶、中成药） | 13% | 3% (2027.12.31前可减按1%) |
| （3）销售非农产品（如种植土等容易与农产品混淆的产品） | 13% | 3% (2027.12.31前可减按1%) |
| （4）销售初加工大米连同包装物 | 9% | 3% (2027.12.31前可减按1%) |

注：区分初级农产品与深加工产品是关键。例如：毛茶（9%/免税），精制茶（13%）；鲜奶（9%/免税），酸奶、调制乳（13%）；中药材（9%/免税），中成药（13%）。
