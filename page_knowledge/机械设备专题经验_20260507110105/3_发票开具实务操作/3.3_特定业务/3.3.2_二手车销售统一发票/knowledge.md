3.3.2. 二手车销售统一发票

## 子目录概览

- **3.3.2.1_开票主体**: 3.3.2.1. 开票主体
（1）二手车交易市场：针对个人与个人、个人与企业之间的交易，由市场开具。
（2）二手车经销企业：收购并销售二手车时，由经销企业自行开具。
- **3.3.2.2_发票联次**: 3.3.2.2. 发票联次
（1）二手车销售统一发票的纸质发票共五联：存根联、记账联、入库联（开票方留存）；发票联、转移登记联（购车方用于记账和过户）。
（2）销售方（企业）无法取得此发票的记账联，应凭自己开具的增值税发票记账联入账，或留存二手车发票复印件。
- **3.3.2.3_反向开具**: 3.3.2.3. 反向开具
已备案汽车销售企业从自然人处购进二手车，允许企业反向开具二手车销售统一发票，用于办理转移登记。
- **3.3.2.4_金额不一致**: 3.3.2.4. 金额不一致
二手车发票金额与增值税专用发票金额不一致时，以增值税专用发票金额为准进行抵扣和入账，二手车发票仅用于过户。
- **3.3.2.5_红字发票开具**: 3.3.2.5. 红字发票开具
当二手车交易市场仅作为开票方，为实际发生二手车交易的购销双方开具纸质红字二手车销售统一发票、带有“二手车销售统一发票”字样的红字数电发票时：
（1）如果销售方为已办理税务信息确认的单位或个体工商户，购买方未进行入账确认的，由销售方填开并提交《红字信息确认单》（以下称“《确认单》”）后，由开票方（二手车市场）开具红字发票。
