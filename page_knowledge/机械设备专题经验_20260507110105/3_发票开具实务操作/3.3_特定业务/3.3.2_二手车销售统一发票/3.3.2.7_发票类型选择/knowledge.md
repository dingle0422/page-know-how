## 本章节内容

3.3.2.7. 发票类型选择
（1）一般纳税人购买设备用于可抵扣项目： 应索取增值税专用发票。
（2）小规模纳税人购买设备： 索取普通发票即可，无需专票（不能抵扣）。
（3）销售给自然人： 只能开具普通发票（或不开票做无票收入），不得开具专用发票。
（4）简易计税项目（3%减按2%）：2026年1月1日前只能开具普通发票。若放弃减税按3%计税，可开具专用发票；2026年1月1日起，纳税人可以开具2%的增专用发票。
