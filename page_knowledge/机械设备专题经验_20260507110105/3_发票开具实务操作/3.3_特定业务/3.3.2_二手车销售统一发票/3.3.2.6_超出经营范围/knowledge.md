## 本章节内容

3.3.2.6. 超出经营范围
企业发生应税行为，即使超出营业执照经营范围，也可以据实开具发票。企业销售机械设备等货物，如果不属于自产农产品、成品油等特定业务，不需要和税局额外进行申请，可以直接按照相应的税收分类编码开具发票。
操作建议：
（1）临时性业务： 确认开票系统中是否有对应品目和税率，如有可直接开具；若无，需向主管税务机关申请核定征收品目。
（2）长期性业务： 建议及时变更营业执照经营范围，再到税务机关办理税务信息变更。
